# Dia Consciente

Um guia simples para ajudar pessoas a entender que emoções são temporárias
e que ações de hoje constroem o amanhã.

## Objetivo
Reduzir o ruído mental e incentivar decisões conscientes,
gerando paz no presente e orgulho no futuro.

## Tecnologias
- React
- JavaScript
- LocalStorage
- Deploy: Render

## Status
MVP em desenvolvimento.
